# Data Analytics Assignment Codes

## Overview
This repository contains Python programs for common Data Analytics laboratory assignments.
Each program is independent and can be executed in Google Colab or Jupyter Notebook after uploading the required dataset.

## Contents

### 1. insurance_dataset.py
**Dataset:** `insurance.csv`

**Topics Covered**
- Data loading and preprocessing
- Handling missing and duplicate values
- Encoding categorical variables
- Scatter Plot
- Histogram with KDE
- Correlation Matrix
- Heatmap Visualization

---

### 2. hr_dataset.py
**Dataset:** `HR_comma_sep.csv`

**Topics Covered**
- Mean
- Median
- Mode
- Range
- Variance
- Standard Deviation
- Interquartile Range (IQR)
- Summary Table
- Histogram
- Boxplot
- Q-Q Plot
- Pair Plot

---

### 3. wine_dataset.py
**Dataset:** `wine.csv`

**Topics Covered**
- Euclidean Distance
- Manhattan Distance
- Minkowski Distance
- Distance Matrix
- Heatmap
- Distance Comparison Plot

---

### 4. nutrient_dataset.py
**Dataset:** `nutrient.csv`

**Topics Covered**
- Data Categorization using `qcut`
- Simple Matching Coefficient (SMC)
- Binary Attribute Transformation
- Jaccard Similarity
- Similarity Comparison

---

## Requirements

Install the required Python packages:

```bash
pip install pandas numpy matplotlib seaborn scipy
```

## Running the Programs

1. Open Google Colab or Jupyter Notebook.
2. Upload the required dataset when prompted.
3. Run the Python script.
4. View the generated tables and visualizations.

## Required Datasets

- insurance.csv
- HR_comma_sep.csv
- wine.csv
- nutrient.csv

Place the datasets in the same working directory as the scripts, or upload them in Google Colab.
